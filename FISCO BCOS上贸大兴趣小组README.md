FISCO BCOS上贸大兴趣小组由多所院校对区块链感兴趣的学生自发成立，小组致力于围绕FISCO BCOS开源社区开源项目进行研究和应用研发，例如基于WeIdentity方案进行基础设施研发与应用研发，完善FISCO BCOS区块链开发工具箱等等。
